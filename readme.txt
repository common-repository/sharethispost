== Description ==

The ShareThisPost Wordpress plugin another tool to provide your readers to share your post to many bookmarking sites like digg, twitter, delicious, technorati, stumbleupon, reddit and cypherbox. Just simply activate the plugin and it will automatically adds the bookmarking options below your post.

To check out the ShareThisPost plugin for WordPress, visit: http://www.cypherbox.net/blog/sharethispost-wordpress-bookmarking-plugin/

Developer's blog, http://www.christophercaralos.com/

Plugin's site, http://www.cypherbox.net/

== Installation ==

1. Unzip the files 
2. Upload sharethispost folder (it contains sharethispost.php and images folder) to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
